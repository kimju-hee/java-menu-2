# java-menu

## 구현 기능

- [ ] **1. 메뉴 파일 정리하기**
  - 카테고리 / 메뉴 이름 순으로 정리하기

<br>

- [ ] **2. 코치 이름 입력 받기**
  - , 로 구분
    - **예외 처리**
      - 코치 이름은 최소 2글자, 최대 4글자
      - 코치는 최소 2명, 최대 5명
      - 이름이 한글이 아닐 경우

<br>

- [ ] **3. 코치별로 못 먹는 음식 입력 받기**
  - ,로 구분
    - **예외 처리**
      - 한글이 아닐 경우
      - 최소 0개, 최대 2개
      - 존재하지 않는 메뉴일 경우
      - 없는 경우 빈칸 입력
    - 코치 객체에 추가

<br>

- [ ] **4. 메뉴 추천**
  - 1) 카테고리 고르기
    - 이미 2번 나온 카테고리면 다시 뽑기
    - 뽑기는 `Randoms.pickNumberInRange()` 사용
    -  1이면 일식, 2면 한식, 3이면 중식, 4면 아시안, 5면 양식
  - 2) 메뉴 고르기
    - 고른 카테고리에 맞는 메뉴들 리스트로 가져오기
    - `Randoms.shuffle()`로 섞은 리스트 처음 값 가져오기
    - 추천할 수 없는 경우 다시 뽑기
      - 이미 먹은 메뉴거나 먹지 못하는 메뉴일 경우

<br>

- [ ] **5. 결과 출력하기**
  - 구분
  - 카테고리
  - 코치 이름, 메뉴
  - 순으로 작성하기
